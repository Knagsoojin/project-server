# 예시 폴더 구조 (3계층 구조)

```
(root)
ㄴ /db
|  ㄴ /models
|  |  ㄴ user-model.js
|  ㄴ /schemas
|  |  ㄴ user-schema.js
|  ㄴ index.js
ㄴ /middlewares
|  ㄴ login-middleware.js
ㄴ /routers
|  ㄴ user-routers.js
ㄴ /services
|  ㄴ user-service.js
ㄴ app.js
```

# 프로젝트 시작하기

Git repository 클론 받기

```
$ git clone https://kdt-gitlab.elice.io/sw_track/class_07/web_project/team10/elice-motors-server.git
```

클론 받은 폴더로 이동

```
$ cd elice-motors-server
```

npm node module설치

```
$ npm install
```

서버 구동

```
$ npm start
```
